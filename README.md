# 🚁 Surveillance Drone System

A drone-based surveillance system designed to monitor large areas from the air using an onboard camera.

## Features

- Aerial surveillance
- Real-time monitoring
- GPS location tracking
- Drone motor control

## Hardware Components

- Drone Frame
- Brushless Motors
- ESC (Electronic Speed Controller)
- Flight Controller
- Li-Po Battery
- Propellers
- FPV Camera Module
- GPS Module
